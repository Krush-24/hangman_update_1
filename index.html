Hangman - Guess the Word

  1. Select the Category from the below given options
  2. Click on submit Button
  3. It will randomly pick one word from the selected category and you need to guess the word
  4. There will keyboard option to select the letters
  5. If you select the correct letter, it will print the letter in the dash
  6. If you select a wrong letter, then there is a hang rope which will have below levels
    • Hat
    • Head
    • Body
    • Right arm
    • Left arm
    • Right leg
    • Left leg
  7. Above the 7 chances you can save the person from hanging by finding the word correctly
